Description of stylistic choices:


Font Choice: I used Arial, sans-serif as my choice of font which is a contemporary sans-serif design. Due to open apertures, it looks natural and is very clear and readable.

Sizes of Heading and Main Text:
  • "h2" header- elements have a font size of 24px, with a bottom margin of 10px and a border at the bottom.
  • "h3" header elements have a font size of 20px, with a bottom margin of 5px.
  • "p" elements have a font size of 16px, with a bottom margin of 10px.
  • The "experience" class is applied to the section element and has a bottom margin of 20px.

Color Choices:
  • Background color: #f4f4f4 (light gray)
  • Header background color: #3498db (blue)
  • Header text color: #fff (white)
  • Border color for headings: #3498db (blue)
  • Text color for main content: #000 (black)
  • Link color: #0000ff (blue)
  • Footer background color: #3498db (blue)
  • Footer text color: #fff (white).

Margin or Padding Choices:
  • Margin: body: 20px; Header: padding: 10px; Sections (excluding the last one): margin-bottom: 20px; Paragraphs within sections: margin-bottom: 10px; List items (li): margin-bottom: 5px.
  • Padding: Body: padding: 20px; Header: padding: 10px; Sections: padding: 0; (unless you want to add specific padding to certain sections) Contact and Experience sections: padding: 0; (unless you want to add specific padding to them).
