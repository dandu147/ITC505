Addendum

In this addendum I explained some of the most common tags that I used, along with that I uesd some styling tags as well to design my web page as per my requirement. Detailed description of used HTML tags are:


  • html- The root element of an HTML document. All other HTML elements are nested within this tag.
  • header- Typically used to define a header for a document or a section. It can contain headings, paragraphs, navigation menus, etc.
  • footer- Used to define a footer for a document or a section. It often contains metadata, copyright information, and links to related documents.
  • script- Used to embed or reference external JavaScript code within an HTML document.
  • body- Represents the content of an HTML document. It contains all the visible content such as text, images, links, etc.
  • p- Stands for "paragraph" and is used to define a paragraph of text.
  • div- Stands for "division" and is a container that is often used to group other HTML elements and apply styles or scripting to them.
  • br- This tag is used to break the line.
  • li- This tag is used to list the items.

  • section class: It is often used to divide a web page into sections or chapters, helping to structure the content and When combined with the class attribute, it becomes a powerful tool for styling and targeting specific sections of your webpage in CSS.
  • style: I used this tag to define my styles directly within the HTML document.
